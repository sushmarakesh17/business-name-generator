# Business Name Generator

## Overview

Business Name Generator is a Python-based application that helps users generate creative and unique business name ideas. The project combines user inputs with predefined words to suggest potential brand names.

## Features

* Generate unique business name suggestions
* Accept user-defined keywords
* Create random word combinations
* Simple command-line interface
* Quick brand name brainstorming tool

## Technologies Used

* Python
* Random Module

## How It Works

1. User enters keywords related to their business.
2. Python processes the input.
3. The application generates multiple business name suggestions.
